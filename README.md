# Credica

A next-generation premium financial landing page, built with pure HTML, CSS, and JS using modern web design principles like Glassmorphism, CSS 3D transforms, and fluid animations.

## Features
- Glassmorphism user interface
- 3D Interactive Mockup 
- Intersection Observer scroll animations
- Fully responsive design

## Setup
Simply open `index.html` in your browser. No build steps required.
